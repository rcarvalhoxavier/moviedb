# MovieDB

MovieDB is a Java Program to organize our movies with a little help from imdb.

 - Access imdb's information directly 
 - Get real movie's name 
 - Preview posters 
 - Rename folders automatically 
 - Quick search

# Development
  
	using Netbeans,
	SQLite,
	JPA + Hibernate
